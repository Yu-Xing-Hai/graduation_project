| 通道名称 | 详细内容设计 |
| :--- | :--- |
| 通道一：综合诊疗决策通道<br>*(核心大脑：负责定方案)* | ① 库里存什么：<br>存“临床场景 -> 推荐方案”的映射规则。<br>*(例：存“NSAIDs治疗无效的活动期患者 -> 推荐生物制剂”；存“单纯外周关节炎 -> 推荐柳氮磺吡啶”)*<br><br>② 检索词：<br>构造的“患者临床小结”（由前序步骤生成的综合描述）。<br>*(例：“26岁男性，NSAIDs治疗效果不佳，BASDAI 6分，CRP升高”)*<br><br>③ 检索内容：<br>召回具体的治疗指南片段或路径图。<br>*(例：“根据2024指南，对于NSAIDs疗效不佳且处于活动期的axSpA患者，建议启用TNF-α抑制剂...”)* |
| 通道二：影像特征与解读通道<br>*(视觉专家：看图说话)* | ① 库里存什么：<br>存“影像征象 -> 病理含义”。<br>*(例：存“骶髂关节骨髓水肿 -> 活动性炎症”；存“骨桥形成 -> 结构性损伤/晚期”)*<br><br>② 检索词：<br>多模态模型提取的影像关键词。<br>*(例：“双侧骶髂关节模糊”、“骨髓水肿”)*<br><br>③ 检索内容：<br>该征象对应的医学解释和诊断价值。<br>*(例：“骨髓水肿是活动性骶髂关节炎的特异性征象，提示疾病处于活动期，是启动生物制剂治疗的重要依据。”)* |
| 通道三：生化指标与临床阈值通道<br>*(检验专家：看数据)* | ① 库里存什么：<br>存“指标+数值范围 -> 临床意义”。<br>*(例：存“CRP > 10mg/L -> 炎症活动”；存“ESR > 20mm/h -> 病情活动”)*<br><br>② 检索词：<br>指标名 + 状态。<br>*(例：“C反应蛋白 升高”、“血沉 45”)*<br><br>③ 检索内容：<br>该指标异常背后的临床提示。<br>*(例：“CRP显著升高提示体内存在急性炎症反应，与强直性脊柱炎的疾病活动度高度相关，需警惕病情进展。”)* |
| 通道四：量表评分与活动度分级通道<br>*(评估专家：定级)* | ① 库里存什么：<br>存“分数 -> 严重程度/对策”。<br>*(例：存“BASDAI ≥ 4 -> 高疾病活动度”；存“ASDAS ≥ 2.1 -> 活动期”)*<br><br>② 检索词：<br>量表名 + 分数。<br>*(例：“BASDAI 评分 6分”)*<br><br>③ 检索内容：<br>该分数对应的分级定义及治疗目标。<br>*(例：“BASDAI评分6分（满分10分）属于高疾病活动度，提示患者症状严重，常规治疗可能不足以控制病情，通常需要升级治疗方案。”)* |
| 通道五：临床症状与鉴别通道<br>*(问诊专家：辨症状)* | ① 库里存什么：<br>存“症状 -> 特征性描述/鉴别点”。<br>*(例：存“炎性腰背痛 -> 夜间痛、晨僵>30分、活动后减轻”)*<br><br>② 检索词：<br>患者主诉症状。<br>*(例：“夜间腰痛”、“早晨起床僵硬”)*<br><br>③ 检索内容：<br>该症状的医学定义及指向性。<br>*(例：“夜间痛和晨僵是炎性腰背痛的典型特征，区别于机械性腰痛，高度提示中轴型脊柱关节炎的可能性。”)* |
